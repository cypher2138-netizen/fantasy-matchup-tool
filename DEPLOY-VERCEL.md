# Deploy to Vercel

## Quick deploy
1. Create a new GitHub repository.
2. Upload all files in this folder to the repo root.
3. Sign in to Vercel.
4. Click **Add New Project**.
5. Import the GitHub repository.
6. Accept the default settings.
7. Click **Deploy**.

## Local test
1. `npm install`
2. `npm start`
3. Open `http://localhost:3000`

## Notes
- This package includes both a local Express server and a Vercel serverless wrapper.
- ESPN data is unofficial and may require occasional endpoint or parser updates.
- No database or login is included in version 1.
